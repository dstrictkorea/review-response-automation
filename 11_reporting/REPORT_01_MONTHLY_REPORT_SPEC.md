# Report 01. Monthly Report Spec

## 1. Branch Report

- one file per branch
- domestic Korean, global English
- branch-specific metrics
- high-risk appendix
- action items

## 2. HQ Report

- all branches
- domestic/global comparison
- channel comparison
- operational risk
- automation performance
- month-over-month

## 3. Required Metrics

- total reviews
- average rating
- response rate
- pending count
- high-risk count
- manual publish count
- Google API publish success
- top categories
- unresolved issues

## 4. Report Naming

`ARTE_REVIEW_REPORT_{BRANCH}_{YYYY_MM}_{LANG}.md`

Examples:
- `ARTE_REVIEW_REPORT_AMBS_2026_04_KO.md`
- `ARTE_REVIEW_REPORT_AMNY_2026_04_EN.md`
- `ARTE_REVIEW_REPORT_HQ_2026_04_KO.md`
