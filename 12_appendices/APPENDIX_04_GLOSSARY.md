# Glossary

| Term | Meaning |
|---|---|
| Human-in-the-loop | 사람이 최종 승인하는 구조 |
| Auto-publish | API로 자동 게시 |
| Semi-automation | 수집/초안은 자동, 게시 수동 |
| Manual Paste Desk | 리뷰 복붙 후 AI 초안 생성 화면 |
| Risk Level | low/medium/high/critical |
| Prompt Version | AI 프롬프트 버전 |
| Approval Decision | 사람이 승인/보류/거절한 기록 |
| Audit Log | 모든 행동 기록 |
| Source Payload | API/scraper 원본 데이터 |
| Normalized Review | 공통 스키마로 변환된 리뷰 |
