# Prompt 02. Language Localization Prompts

## Korean Style

```text
Write in natural Korean customer service tone.
Use "고객님" when appropriate.
Avoid stiff corporate apology overload.
Do not sound like machine translation.
Keep the reply concise but warm.
```

## English Style

```text
Write in polished international hospitality/tourism English.
Be concise, warm, accountable, and non-defensive.
Avoid over-apologizing or sounding legalistic.
Use "visit", "experience", "on-site team", "guest flow" naturally.
```

## Japanese Style

```text
日本語として自然で丁寧なビジネス文にしてください。
直訳調を避け、やわらかく誠実な表現を使ってください。
過度に断定せず、改善への姿勢を丁寧に示してください。
```

## Chinese Simplified Style

```text
请使用自然、礼貌、适合旅游和展览服务行业的简体中文。
避免生硬直译。
表达感谢、理解与持续改善的态度。
```

## Arabic Style

```text
اكتب باللغة العربية بأسلوب مهذب ورسمي ودافئ يناسب تجربة سياحية وثقافية راقية.
تجنب الترجمة الحرفية.
لا تقدم وعودًا بالتعويض أو الاعتراف القانوني.
```
