| Chinese Simplified | Chinese reviews unless traditional detected | 자연스러운 관광/서비스 응대체 |
| Arabic | Arabic reviews, Dubai priority | 격식 있고 정중한 표현 |

## 3. Language Selection Rule

1. Detect review text language.
2. If language is KO/EN/JA/ZH/AR, reply in same language.
3. If unclear, use branch default language.
4. If mixed language, use language of complaint content.
5. Do not infer nationality from name.
6. Do not machine-translate literally.
