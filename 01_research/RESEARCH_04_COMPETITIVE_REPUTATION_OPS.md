# Competitive Reputation Operations Notes

## 1. What Good Review Ops Looks Like

Good brand review operations do not try to win arguments. They do four things:

1. Acknowledge the customer.
2. Protect the brand tone.
3. Show operational listening.
4. Move sensitive matters out of public comment.

## 2. Museum/Tourism Context

ARTE MUSEUM reviews are likely to mention:

- visual beauty
- immersion
- ticket price
- crowding
- waiting line
- children/family experience
- photo opportunities
- staff guidance
- wayfinding
- accessibility
- refund/ticketing
- language support
- expectation mismatch
- sensory discomfort
- cafe/gift shop
- parking/transportation

## 3. Response Strategy by Review Type

| Review Type | Strategy |
|---|---|
| Beautiful exhibition | gratitude + future invite |
| Crowded | acknowledge + improve visitor flow |
| Expensive | value framing without arguing |
| Staff complaint | apologize + internal review, no names |
| Child/family issue | extra care and escalation |
| Accessibility issue | acknowledge seriously + route |
| Refund/ticket issue | public empathy + private support channel |
| Photo complaint | explain experience carefully if needed |
