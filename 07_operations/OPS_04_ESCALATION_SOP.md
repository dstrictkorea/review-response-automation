# OPS 04. Escalation SOP

## Escalation Targets

| Issue | Owner |
|---|---|
| refund/ticketing | CS Manager |
| safety/accident | Branch Manager + HQ Ops |
| child safety | Branch Manager + HQ Ops immediately |
| legal/police/media | HQ Ops + Legal |
| staff complaint | Branch Manager |
| discrimination | HQ Ops |
| privacy/CCTV | System/Operations Manager |

## Escalation Message Format

- branch
- channel
- rating
- review URL
- review text
- AI summary
- risk reason
- recommended action
- SLA
