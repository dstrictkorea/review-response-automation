# OPS 03. Manual Publishing SOP

## Channels

- TripAdvisor
- Naver
- Domestic OTA
- Marketplace channels

## Steps

1. Open platform admin page manually.
2. Find target review.
3. Copy approved reply from dashboard.
4. Paste into platform.
5. Submit.
6. Return to dashboard.
7. Mark manual published.
8. Add screenshot/reference if required.

## Manual Publish Must Not

- modify approved reply without recording
- publish high-risk reply without escalation
- share platform credentials
- use browser automation to bypass restrictions
