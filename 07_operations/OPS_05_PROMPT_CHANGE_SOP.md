# OPS 05. Prompt Change SOP

## Steps

1. Identify issue.
2. Update prompt draft.
3. Run regression test set.
4. Compare before/after.
5. Review with operations owner.
6. Version prompt.
7. Deploy to staging.
8. Monitor production for 7 days.

## Do Not

- edit prompt directly in production n8n node
- remove safety constraints for convenience
- change language rules without QA
