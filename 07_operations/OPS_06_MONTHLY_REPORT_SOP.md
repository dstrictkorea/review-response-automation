# OPS 06. Monthly Report SOP

## Timeline

- 1st day 09:00: generate draft reports
- 1st day 13:00: system/ops review
- 2nd day: branch manager review if needed
- 3rd day: HQ distribution

## Checks

- date range
- branch list
- channel coverage
- review counts
- high-risk list
- unresolved replies
- evidence labels
- action items

## Distribution

- Domestic branch reports: Korean
- Global branch reports: English
- HQ summary: Korean
