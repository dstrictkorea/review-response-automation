# 02. Non-Negotiable Policy Lock

## 1. 절대 금지

| 금지 항목 | 이유 |
|---|---|
| 무승인 대량 자동 게시 | Google/플랫폼 리스크, 브랜드 리스크 |
| 가짜 리뷰 작성 | 정책 위반 및 신뢰 훼손 |
| 리뷰 보상/리뷰 유도 자동화 | 리뷰 조작으로 판단될 수 있음 |
| CAPTCHA/로그인 우회 스크래핑 | 플랫폼 제한 우회 및 계정 차단 리스크 |
| 고객 국적 추정 | 차별/오판/어색한 응대 리스크 |
| 직원 실명 공개 답변 | 개인정보/노무/법무 리스크 |
| CCTV 확인 언급 | 개인정보·보안 리스크 |
| 환불·보상 약속 | 권한 없는 약속/분쟁 확대 리스크 |
| 법적 책임 인정 | 법무 리스크 |
| 모든 부정 리뷰에 동일 문장 반복 | 스팸/저품질/브랜드 저하 리스크 |

## 2. 반드시 유지

| 유지 항목 | 내용 |
|---|---|
| 관리자 승인 | Google 포함 모든 게시 전 승인 |
| 위험도 라우팅 | low/medium/high/critical |
| 언어 감지 | 리뷰 본문 기준 |
| 답변 이력 | 원문, 초안, 수정본, 승인자, 게시 상태 저장 |
| 감사 로그 | 수집, 생성, 수정, 승인, 게시, 실패 전부 기록 |
| 중복 방지 | 동일 리뷰 재수집/중복 게시 차단 |
| rollback | Google 답변 삭제/수정 가능성 고려 |
| 월간 리포트 | 운영 개선 자료로 자동 생성 |

## 3. 자동 게시 후보 조건

자동 게시는 Phase 3 이후에만 검토한다. 초기 MVP에서는 모든 답변이 승인형이다.

자동 게시 후보는 아래 조건을 모두 만족해야 한다.

1. Google 리뷰다.
2. 4~5점이다.
3. 안전/사고/환불/직원/차별/아동/개인정보/법무 키워드가 없다.
4. AI가 언어를 0.95 이상 확신한다.
5. 답변 품질 평가 점수 0.90 이상이다.
6. 최근 30일 동안 해당 지점 답변 패턴 반복도가 임계값 미만이다.
7. 운영 책임자가 해당 지점 자동 게시를 명시 승인했다.

## 4. 자동 게시 금지 조건

1. 1~3점 리뷰
2. 별점은 높지만 민감 키워드 포함
3. 고객이 구체적 피해/불편/환불 요구
4. 직원 실명/외모/태도 직접 언급
5. 사고, 부상, 아동 안전, 위생, 분실, 도난, 차별, 장애 접근성, 법적 조치
6. 언어 감지 불확실
7. 번역이 어려운 풍자/비꼼/속어
8. 스크래핑/수동 입력으로 들어온 채널
